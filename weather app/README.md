# 🌦️ Weather App

A simple and elegant weather application built using HTML, CSS, and JavaScript. This app allows users to search for real-time weather information of any city using the OpenWeatherMap API.

## 🔍 Features

- 🌐 Search for any city's weather
- 🌡️ Displays current temperature in Celsius
- 💨 Shows humidity and wind speed
- ⛅ Weather icons change dynamically based on the condition
- ⚠️ Error handling for invalid or unavailable city names

## ⚠️ Limitations
Some cities might not be found due to the limitations of the free API tier.

The app currently supports only metric units (°C, km/h).
