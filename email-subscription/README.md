# Form to Google Sheets Project

This project allows you to **submit form data from a web page directly into a Google Sheet**. It is fully responsive and includes smooth input animations for better user experience.  

The project was inspired by and built with guidance from [Jamie Wilson's Form to Google Sheets](https://github.com/jamiewilson/form-to-google-sheets).

---

## Features

- Responsive form inputs that scale on mobile, tablet, and desktop screens
- Smooth fade-in/fade-out animation for confirmation messages
- Google Sheets integration to store submitted form data
- Modern styling with CSS `clamp()`, flexible widths, and `box-sizing` fixes
- Fully mobile-friendly with pre-modern `@media` queries
